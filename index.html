Welcome!

Welcome to my corner of the internet! I'm a full-stack web developer with the experience to build, modify, and manage websites. I've built databases using MySQL and MongoDB. I've blended HTML, CSS, and Javascript to create atmospheric, responsive websites. I'm constantly on the lookout for emerging technologies and new ways to use old ones.


As a former educator, I managed roughly 150 students each year as I created and modified a year-long curriculum for them. In that time I also mastered the art of public speaking and presenting complex concepts to individuals unfamiliar with the material.

Brews n Beans

App using API's and firebase to allow users to search for and rate coffee shop locations. Firebase is able to receive information from the apis to generate location unique likes and dislikes.

Horror Trivia

Horror themed survival trivia game. Powered by javascript setTimeouts, this page cycles through an array of questions, modifying content based on user responses and time remaining.

Monopoly

Full-Stack game using CSS grid, authentication, and a MySQL database. Database put and get request drive functionality while socket.io causes updates to happen live on multiple pages at once.

Javascript RPG

A javascript intensive take on classical rpgs. User customizes their party and uses abilities and a functioning inventory to defeat waves of randomly generated foes.
